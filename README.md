# rjthandi.dev

Personal website combining the best of:
- **Adam Majmudar** (adammaj.com) — narrative "about me", interests, clean structure
- **Aidan Andrews** (aidanandrews.info) — developer portfolio, projects, writing

## Run locally

Open `index.html` in a browser, or use a simple server:

```bash
npx serve .
```

## Customize

1. **About & interests** — Edit the bullet points in `index.html` under "Some things about me" and "Some things I'm interested in"
2. **Projects** — Replace the placeholder projects with your own (name, description, link)
3. **Writing** — Add real blog posts or link to Substack/Medium
4. **Social links** — Update GitHub, Twitter, LinkedIn URLs in the sidebar
5. **Colors** — Change `--accent` and `--bg` in `styles.css` for your palette

## Deploy

- **Vercel**: `npx vercel`
- **Netlify**: Drag the folder to netlify.com/drop
- **GitHub Pages**: Push to a repo, enable Pages in Settings

Point your domain `rjthandi.dev` at the deployment.
