Projects

  • FieldScout

    Field data collection and mapping for conservation work, uses OpenCV and Python to detect and track animals. The goal was to make it easier for Cache Creek Conservancy to monitor their land and animals.

  • DoorDetector

    Computer vision project for door detection; uses an Arduino UNO3 to communicate Serial output through C++ -> Python and tweeting the result.

  • FreightBite

    Winner of SacHacks VII, FreightBite is a web application that allows owner operators to track their fleet and manage workload to meet HOS optimization. It was built with React, Node.js, and PostgreSQL.

  • dAdvisor

    Winner of ProdCon 26'. A web app for UC Davis undergraduates to plan a major change: view requirements for a target major, mark completed courses, plan quarters, and estimate time to graduation. Built with React, TypeScript, and Vite.
